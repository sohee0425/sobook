## 프로젝트소개 
'SOBOOK'은 사용자들이 원하는 도서를 구매할 수 있는 쇼핑몰입니다.

이 프로젝트는 Spring Boot 기반으로 개발되었으며, 사용자와 관리자 페이지를 통해 책 구매, 상품관리, 주문 및 리뷰관리를 할 수 있습니다.

## 개발환경
+ Front :  HTML, CSS, Boot-strap, JavaScript, jQuery, Thymeleaf
+ Back-end : SpringBoot, Mybatis
+ Database : MySQL
+ 버전 및 이슈 관리 : GitHub
+ API : 카카오 페이, 카카오 우편번호

## 페이지별 기능
### [사용자]

#### [회원가입]
+ 아이디 중복체크로 유효성 검사가 진행되고, 통과하지 못한 경우 경고 문구가 표시됩니다.
+ 이메일 인증코드가 발급되고, 인증 코드 일치시 유효검사가 통과되고 불일치 시 경고 문구가 표시됩니다.
+ 비밀번호 불일치 시 경고 문구가 하단에 표시됩니다.
+ 카카오 우편번호 API를 사용하여 우편번호 검색이 가능합니다.
+ 필수항목을 입력하지 않은 경우 회원가입 처리 되지 않도록 유효성 검사가 진행됩니다.
+ 유효성 검사가 통과된 경우 로그인 페이지로 이동합니다.

#### [로그인]
+ 아이디와 비밀번호를 입력 후 로그인 버튼 클릭시 유효성 검사가 진행됩니다.
+ 로그인 버튼 클릭 시 아이디 또는 비밀번호가 일치하지 않을 경우 경고 문구가 나타나며, 로그인 성공 시 홈 화면으로 이동합니다.

  #### [아이디, 비밀번호 찾기]
+ 아이디 찾기 : 회원가입 시 입력했던 이름과 이메일 일치 시 아이디가 이메일로 전송됩니다.
+ 임시 비밀번호 발급 : 회원가입 시 입력했던 아이디와 이메일 일치 시 임시 비밀번호가 이메일로 전송됩니다.
 
#### [로그아웃]
+ 메인화면 상단에 로그아웃 버튼을 클릭하면 로그아웃이 됩니다.
+ 로그아웃 된 상태에서는 상품 주문, 장바구니 등 기능이 제한됩니다.

#### [마이페이지]
##### [회원정보수정]
   + 입력란에 수정 정보를 입력 후, 수정 버튼 클릭시 회원정보가 수정됩니다.
##### [비밀번호 변경]
   + 기존 비밀번호와 신규 비밀번호를 입력합니다.
   +  기존 비밀번호와 db에 저장된 비밀번호가 일치한지 확인합니다. 불일치 시 비밀번호 변경이 불가합니다.
   +  신규 비밀번호와 신규 비밀번호 확인이 불일치 시 비밀번호 변경이 불가합니다.
   + 일치 시, 사용자에게서 받은 신규 비밀번호를 암호화하여 Database에 저장합니다.
##### [회원탈퇴]
   + 비밀번호를 입력 후 탈퇴 버튼 클릭 시 회원탈퇴가 됩니다.
   + 사용자가 입력한 비밀번호와 db에 저장된 비밀번호가 일치하면 db에 저장된 회원정보를 삭제합니다.
##### [주문내역]
+ 사용자가 주문한 내역을 목록으로 보여줍니다.
+  주문 번호 클릭 시 해당 주문내역 상세정보가 나옵니다.
++ 상세정보에서 배송지 변경이 가능합니다.
##### [장바구니]
+ 사용자가 장바구니에 담아놓은 목록을 보여줍니다.
+ 수량 입력란에서 수량 변경이 가능합니다.
+ 해당 상품 체크박스 클릭으로 선택 삭제가 가능합니다.
+ 장바구니 비우기 버튼으로 장바구니 목록 전체 삭제가 가능합니다.
+ 주문하기 버튼 클릭 시 장바구니에 담긴 상품이 주문 페이지로 이동하여 주문이 가능합니다.
##### [나의 리뷰]
+ 구매목록에서 리뷰를 작성할 수 있습니다.
+ 배송완료 시에만 리뷰 작성할 수 있습니다.
+ 수정/삭제 할 수 있도록 구현하였습니다.
##### [위시 리스트]
+ 사용자가 찜한 찜 목록을 보여줍니다.
+ 찜 상품 개별적으로 장바구니 담기, 주문하기, 삭제가 가능합니다.
+ 상품 이미지 클릭 시 상품 상세 정보 페이지로 이동합니다.
+ 수량 변경하여 장바구니 담기 및 주문이 가능합니다.
+ 체크 박스로 선택하여 장바구니 담기 및 삭제가 가능합니다.

#### [책 카테고리]
+ 1차 카테고리 항목 중 하나 클릭 시, 2차 카테고리가 표시됩니다.
+ 2차 카테고리 클릭 시, 해당 카테고리 목록 페이지로 이동합니다.

#### [책 정보]
##### [책 목록]
+ 선택한 카테고리에 해당하는 책 정보들이 목록으로 표시됩니다.
+ 장바구니 혹은 장바구니 클릭 시 해당 페이지로 이동할 수 있습니다.
+ 책 이미지 클릭 시 해당 책 상세 페이지로 이동합니다.

##### [책 상세페이지]
+ 해당 책 상세정보가 표시됩니다.
+ 해당 책 리뷰가 표시됩니다.
  + 해당 사용자가 입력한 리뷰를 수정/삭제 할 수 있습니다.
+ 장바구니, 주문하기, 찜 기능이 구현되어 있습니다.
  
#### [상품주문 및 결제기능]
+ 주문을 선택한 상품들이 표시 됩니다.
+ 주문자 정보와 다른 배송지를 선택할 경우 배송지를 새로 작성헐 수 있습니다.
+ 카카오페이 API를 활용하여 결제 시스템을 구현하였습니다.
+ 무통장 입금 선택 시 입력란에 정보를 입력 후 결제하기 버튼을 클릭하면 주문완료 페이지로 이동합니다.
+ 주문 완료 시 이메일이 발송됩니다.

---

### [관리자]
#### [로그인]
+ 아이디와 비밀번호를 입력 후 로그인 버튼 클릭시 유효성 검사가 진행됩니다.
+ 로그인 버튼 클릭 시 아이디 또는 비밀번호가 일치하지 않을 경우 경고 문구가 나타나며, 로그인 성공 시 관리자 홈 화면으로 이동합니다.

#### [상품관리]
##### [상품등록]
+ 정보를 입력 후, 등록 버튼 시 상품이 등록됩니다.
+ ckEditor를 활용하여 설명란에 이미지, 글 편집 기능을 추가했습니다.
##### [상품 목록]
+ 상품 내용 수정/삭제가 가능합니다.
+ 상세 조건에 따른 상품 검색 가능합니다.
#### [주문관리]
+ 사용자가 주문한 내역들을 관리합니다.
+ 사용자의 결제 상태 및 배송 상태에 따라 상태를 변경할 수 있습니다.
+ 사용자가 주문을 취소/반품/교환 요청할 경우 변경할 수 있습니다.
+ 상세 조건에 따른 주문 내역이 검색 가능합니다.
+ 주문 번호 및 상세 정보 버튼 클릭 시 주문 상세내역 페이지로 이동합니다.
+ 주문 상세내역 페이지에서 주문 상품 삭제, 결제/배송지 변경이 가능하며 관리자 메모 기능으로 특이사항 작성이 가능합니다.
#### [리뷰관리]
+ 사용자가 작성한 리뷰에 답변을 할 수 있습니다.
+ 리뷰 답변/수정이 가능합니다.
+ 상세 조건에 따른 리뷰 검색이 가능합니다.
#### [게시글(공지사항)관리]
+ 공지사항 등록 및 수정이 가능합니다.

## 프로젝트 후기
개인으로 프로젝트를 시작하면서 첫 번째 목표는 직업훈련 때 배운 기술들을 프로젝트에 다 사용해보는 것이었습니다. 처음에는 단순히 기능을 구현하는 데 집중했지만 점점 코드들이 눈에 익으면서 기술적으로 필요함을 생각하게 되는 생각의 첫 확장으로 이어졌습니다.
두 번째 목표는 오류를 받아들이고 기록해두기였습니다. 처음부터 끝까지 혼자 쇼핑몰을 제작하면서 예상하지 못한 오류들을 해결하면서 시간은 많이 사용되었지만, 오류가 눈에 익으면서 오류를 봐도 더 이상 두렵지 않게 되었고 분석하는 습관이 생기며 오류들은 저에게 양분이 되어줬습니다.
이 경험들을 바탕으로 다음 프로젝트를 진행할 때 체계적으로 나아가고자 합니다.
